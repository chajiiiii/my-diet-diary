# My Diet Diary

[📖Click here to check out the app🍕](https://my-dd-da037262feac.herokuapp.com/)

## Description

My Diet Diary is a diet tracking application. You can log your daily meals with time, photos, weight, etc. This is a solo project.

## Technologies

- HTML
- CSS
- JavaScript
- Express.js
- Node.js
- MongoDB
- Mongoose
- EJS
- Passport.js
